# AMAP

#### Description
amap file load and save library

#### Prerequisites
1. cmake >= 3.13
2. gcc/g++ >=5
3. Eigen3

#### Build
```c++
git clone https://gitee.com/IntallianceInnovation/amap.git
cd amap
mkdir build 
cd build
cmake ..
make -j4
```

#### usage
1. used in your project
just include the header files
